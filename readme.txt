== Description ==

Display your top commenters in a widget.

Give your loyal visitors and contributors some recognition by adding a list of your top commenters to your blog.

* List your top commenters with the option to display their Gravatar, and several other options.
* Choose from 2 formats of the top commenter list, with complete control of styles via css.
* Exclude Blog Authors from the top comment list.
* The list uses a cache system for improved performance. List updates only when a comment status is changed, options updated, comment deleted or a new comment is added.

= Extra Feature =
* Add a special Icon next to each of your Top Commenter's name in their comments to give them a little special recognition for being a regular contributor.



= Initial Release =
This is the first release of this plugin. If you come across any bugs or have suggestions or ideas for improving, please send me a message at http://justmyecho.com/2010/07/top-contributors-plugin-wordpress/

* Fixed the bug where User's Gravatar was not showing.

== Installation ==

1. Download and extract top-contributors.zip file.
2. Upload the folder containing the Plugin files to your WordPress Plugins folder (usually ../wp-content/plugins/ folder).
3. Activate the Plugin via the 'Plugins' menu in WordPress.
4. Go to "Settings > Top Contributors" page to set your options and add the widget.
